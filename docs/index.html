Why would I use Argus?

Gain insight into directory and file changes for apps in your Kubernetes cluster; be confident come assessment time.

In more words…

At the time of creating this product, there weren’t any other solutions out there. We designed the architecture around common Linux kernel functionality to ensure well-known practices around inode monitoring. The rest of which was making sure it performed well in a clustered environment, and had features that security experts and assessors could healthily engage on.
